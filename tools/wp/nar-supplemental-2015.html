Online Supplemental Material

This is a temporary demonstration page for the review of the online supplemental material associated with an NAR Database Issue submission: WikiPathways: Capturing the Full Diversity of Pathway Knowledge

According to author instructions, "Authors are encouraged to be innovative in their use of such supplementary data, taking advantage of the fact that electronic material, unlike the printed text, does not need to be static." We are thus providing 3 non-static submissions for supplemental material: 2 embedded videos and an interactive pathway widget, each based on simple HTML-embed code (also provided below).

Interactive Pathway Widget

Supplemental 3. An interactive pathway viewer that can be embedded into any website. The widget includes pan and zoom, search, and clickable datanodes with popup annotation panels. The view can be set to any width and height, and customized to highlight particular datanodes with any color. Full documentation can be found at http://wikipathways.org/index.php/PathwayWidget.
